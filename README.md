# МЯТА FUTURE LAB · Проект4

Сайт-приглашение на слёт партнёров сети «Мята» · **27 августа 2026**, мультимедийное пространство Connect, Москва. Свёрстан 1-в-1 по дизайн-макету «Desktop - 2» (светлое серебро, чёрный, голо-градиенты, строгие линии).

**Живой сайт:** https://myatnyi-vibe.github.io/myata-future-lab4/
**Проект3 (светлый воздушный, three.js):** https://myatnyi-vibe.github.io/myata-future-lab3/

## Что внутри

- Секции по макету: hero (гироскоп, кому/от), чёрная бегущая строка,
  01 КОГДА (календарь-таблица), 02 ГДЕ (CONNECT) — далее по структуре
  Проекта3 в стиле макета: 03 ДРЕСС-КОД, 04 ПРОГРАММА, 05 ВОСПОМИНАНИЯ, финал
- Живая графика (three.js 0.160 с cdn.jsdelivr, importmap):
  - js/scene.js — хромовый 3D-гироскоп в герое (PMREM-окружение из
    рисованной equirect-карты, ирисцентный металл); фолбэк — assets/gyro.webp
  - js/veil.js — «шёлковая» вуаль из ~18k GPU-точек в секции «Когда»
    (кастомный шейдер); фолбэк — 2D-canvas версия в main.js
  - оба канваса переживают потерю WebGL-контекста и уважают reduced-motion
- Шрифт ALS Granate (вариативный woff2), ассеты извлечены из ПДФ-макета
- Форма заявки → Google Таблица «заявки на слёт» (общий Apps Script
  веб-хук всех версий; источник различается колонкой «Страница»)
- Чистая статика: HTML + CSS + JS, без сборки
